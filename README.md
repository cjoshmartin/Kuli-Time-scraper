# Kuli-Time-scraper
Using Puppeteer to scrape Indiana University time tracking site. Will return hours worked and clock in/out for the user.

## How To run it: 

* Create a `.env` file with USERNAME, PASSWORD and RATE

```
npm i
npm start
```
